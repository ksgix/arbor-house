# Arbor House | Immersive Archive

An interactive digital experience documenting the **Arbor House** by **Brown & Brown Architects**.

This project serves as a minimalistic, immersive web archive that allows users to explore the architectural layout, photography, and atmosphere of the home through interactive SVG plans and ambient audio.

*   **Immersive Audio:** Cinematic entrance with ambient sound and a custom-built mini player (Volume/Pause controls).
*   **Minimalist UI:** Custom cursor, scroll reveal animations, and magnetic navigation.
*   **Gallery System:** Dynamic masonry grid loading images from local directories.
*   **Cinematic Video:** Integrated Vimeo showcase of the property.
